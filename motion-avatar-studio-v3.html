6. Consejos
Mixamo
Suele funcionar bien con auto-mapeo, pero el editor te deja afinar hombros, brazos y piernas.
Rigs propios
Si los nombres no son estándar, usa el editor manual y guarda el mapping.
Exportación
Para software 3D moderno usa GLB. Para pipelines clásicos de animación usa BVH.
