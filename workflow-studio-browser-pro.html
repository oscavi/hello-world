Workflow Studio Browser Pro

Versión solo navegador, pensada para clasificación de imágenes con TensorFlow.js. Intenta usar GPU automáticamente con prioridad WebGPU → WebGL → CPU. Si la GPU no está disponible o el navegador no la soporta, hace fallback seguro a otro backend. El entrenamiento muestra estado en tiempo real por batch y epoch para que puedas comprobar que está funcionando.
