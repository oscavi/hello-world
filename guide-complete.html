🚀 Guía Completa Vision AI Edge

Guía actualizada del sistema completo: dataset, entrenamiento, exportación, optimización, despliegue versionado, inferencia web adaptativa y dashboard. Esta guía reúne lo histórico y lo último en un solo sitio.

1. Flujo completo

  1. Crear dataset desde móvil con cajas YOLO.
  2. Validar dataset y preparar entrenamiento en PC.
  3. Entrenar detector con YOLO en el repo privado.
  4. Exportar a ONNX.
  5. Cuantizar a INT8 para móvil.
  6. Desplegar versión al repo público.
  7. Gestionar versiones desde el dashboard.
  8. Lanzar inferencia adaptativa en navegador.

3. Qué usar en cada caso

iPhone / Android: crea dataset con yolo-builder-mobile.html y en producción usa model.int8.onnx.

PC para preparar entrenamiento: usa yolo-workbench-pc.html para validar el dataset y obtener recomendaciones de modelo.

Conversión / despliegue web: usa yolo-converter-workbench.html y los scripts del repo privado.

Producción web: usa vision-yolo-adaptive-auto.html. Elige automáticamente entre modelo completo e INT8.

5. Uso recomendado del dashboard

  1. Abre model-dashboard.html.
  2. Pega la raíz de modelos, por ejemplo https://oscavi.github.io/hello-world/modelos/vespas/.
  3. Comprueba la versión activa en latest.json.
  4. Lanza desde el dashboard la app de inferencia con la URL correcta.
  5. Si una versión falla, usa una carpeta versionada anterior.

6. Recomendaciones de uso

  • En móvil usa siempre model.int8.onnx.
  • En PC medio o potente usa model.onnx.
  • Para producción apunta a latest/.
  • Para pruebas y rollback usa carpetas versionadas.
  • Mantén labels.json y config.json junto al modelo.
